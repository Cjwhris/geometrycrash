# how to start
* `git clone https://github.com/LHXnois/geometrycrash.git`
* 用ue4新建一个项目
* 把clone下的文件全部拷贝到项目里覆盖
* 打开`TwinStickBP/Maps/TwinStickBP/Maps/level1`,开始修改或搭建
* 修改完成后，`git pull`+`git push`
